
# SpeedyWorks — Premium Sports Car Service (KOMPATKOWA)

Projekt pokazuje SASS (SCSS) w praktyce: zmienne, zagnieżdżenia, mixiny, dziedziczenie, operatory, funkcje, moduły (@use), responsywność i animacje. Zdjęcia pochodzą z Unsplash (hotlink CDN).

## Uruchomienie (Windows, bez PATH)
```
"C:\Users\llenart\AppData\Roaming\npm\sass.cmd" --watch scss/style.scss css/style.css
```
Otwórz `index.html` w przeglądarce.

## Struktura
```
scss/
  base/ (reset, typography)
  components/ (buttons, cards)
  layout/ (nav, hero, sections, contact)
  utils/ (variables, mixins, functions)
  style.scss
css/style.css (generowany)
index.html
```
